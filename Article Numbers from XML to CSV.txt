Импортируем необходимые модули: requests, xml.etree.ElementTree, csv, os.
Задаем URL для получения XML данных.
Отправляем GET запрос по указанному URL и сохраняем полученные данные в переменную response.
Получаем текст из XML данных.
Создаем корневой элемент для дальнейшего парсинга XML.
Создаем пустой список article_numbers, в котором будем хранить артикулы.
Проходимся по каждому элементу <offer> в XML и ищем элемент <vendorCode>.
Если элемент <vendorCode> найден, то получаем его текст и сохраняем в переменную article_number. Иначе, присваиваем значение "null".
Добавляем артикул в список article_numbers.
Задаем путь для сохранения файла. save_path указывает путь к папке, в которой будет сохранен файл. В данном примере, файл будет сохранен в папку "Doki" в директории "Downloads" пользователя "Solrikk".
Создаем папку для сохранения файла, если она не существует.
Формируем полный путь к файлу.
Открываем файл для записи данных в режиме "w" (write).
Создаем объект writer для записи данных в CSV формате.
Записываем заголовок ("article_number") в файл.
Записываем каждый артикул в отдельную строку файла.
Выводим сообщение об успешном сохранении данных в файл "offer_article_numbers.csv".
